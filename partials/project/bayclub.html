The recently relaunched bayclubs.com brought the company's 11 various websites together on the same platform. The site serves as a portal for club members to view upcoming classes, events, and other information.

My repsonsibility on this site was from top to bottom. The site is built in Umbraco, and I built all of the neccessary types needed to store the different pieces of data. I leveraged Umbraco's API to build custom data uploaders, to migrate their old data from ExpressionEngine into Umbraco. I built a JSON webservice for their class schedules, filter classes by class type, instructor, and location. Finally, I developed all of the page templates.

The result of this project was turnkey - after leading the training sessions, the client has been able to manage all pieces of the site themselves.

C#, Umbraco, XSLT, XPath, custom JSON services, Bootstrap
